1. 创建s3存储桶，  
权限--存储桶策略
```
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Action": [
                "s3:PutObject"
            ],
            "Effect": "Allow",
            "Resource": "arn:aws:s3:::存储桶名称/*",
            "Principal": {
                "AWS": [
                    "ELB账户ID"
                ]
            }
        }
    ]
}

```
